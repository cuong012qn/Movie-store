# Movie-store
Add appsettings.json
```json
{
  "AllowedHosts": "*",
  "Logging": {
    "LogLevel": {
      "Default": "Information",
      "Microsoft": "Warning",
      "Microsoft.Hosting.Lifetime": "Information"
    }
  },
  "Secretkey": "Your secret key",
  "Api": "your host",
  "ConnectionStrings": {
    "MovieDBContextConnection": "Server=Name server;Database=DBName;"
  },
}
```
